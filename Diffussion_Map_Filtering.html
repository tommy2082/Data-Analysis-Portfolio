Diffusion Map Abdominal Fetal Electrocardiogram Filtering¶

Abdominal Fetal Electrocardiogram is recorded by attaching electrodes to the maternal abdominal area. It of?ers a non-invasive way to monitor fetal heart activity. Compared to Direct Fetal Electrocardiogram which requires electrodes to be attached to the fetus. However, the signal from the abdominal fetal Electrocardiogram is contaminated with the pulse of the maternal heart waves. The goal of this project is to filter out fetal heartbeat from abdominal maternal ECG. All signals are sampled at 1 KHz with 16-bit resolution.A diffusion map is a dimensionality reduction technique used in data analysis and machine learning, particularly effective for revealing the underlying manifold structure of high-dimensional datasets. The method, introduced by Coifman and Lafon in 2006, employs the concept of diffusion processes to provide a geometric framework for data representation.

The Electrocardiogram contains two different electrical pulses from both the heartbeat of the mother and the fetus. Thus the signal spikes seem aperiodic. We will separate the two signals using Diffusion Map. We can see 4 large peaks with large periods and 7 smaller peaks with smaller bandwidths and periods. Using Diffusion Map we can select geometric properties such as the size and width of the peaks. Using Diffusion Map we will first isolate the stronger maternal signal and then filter out the fetal ECG signal.

# We begin Diffusion Map with the construction of a Markov chain, 
# where transition probabilities between data points are defined based on their similarity.
gen_P <- function(f,sigma,BandWidth){ 
  P = matrix(0,nrow = length(f)-2* BandWidth ,ncol = length(f) - 2*BandWidth) 
  weight = c(1:(BandWidth+1) , BandWidth:1) / (BandWidth+1)
       for(i in (BandWidth+1):(length(f)-BandWidth)  ){
          for(j in i:(length(f)-BandWidth) ){
            # generating distance using Gaussian Kernel 
              # We will be using a weighted Gaussian kernel based on nearby 
             SS = sum((f[(i-BandWidth):(i+BandWidth)] - 
                         f[(j-BandWidth):(j+BandWidth)])^2 * weight^2)
              P[i-BandWidth,j-BandWidth] = exp(-SS/sigma^2/2)
        }
      }
   P = P + t(P) - diag(rep(2,length(f)-2*BandWidth))
  diag(1/rowSums(P)) %*% P
}
# Selecting the second largest eigenvalue's corresponding eigenvector for smooth filtering
filter_dm <-function(data,Sig,BandWidth){
  P = gen_P(data,Sig,BandWidth)
  dm = eigen(P)
  dm$vectors[,2]
}

# filter for specific bandwidth and inverse precision parameter of the weighted Gaussian kernel
BandWidth = 15
sig = 4
filter1 = filter_dm(data,sig,BandWidth)
# fit the filter to the data set by finding a linear transformation minimizing the square distance between data
lm_coef = lm( data[(1+BandWidth):(length(data)-BandWidth) ]~ filter1 )$coefficients
maternal = lm_coef[1] + lm_coef[2]* filter1
fetal = data[(1+BandWidth):(length(data)-BandWidth) ] - maternal
plot(maternal,type = "l",xlab = " ",main = "Maternal")
